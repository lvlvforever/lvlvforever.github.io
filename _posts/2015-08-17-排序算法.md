###排序依赖的方法和数据结构
####1 交换和比较
1. 数组中两个元素交换是比较简单的，可以使用一个临时元素保存中间值，
空间复杂度是O(1)。

{% highlight java linenos %}
private void swap(int[] nums,int i ,int j){
		if(i == j) return;
		if(nums[i] == nums[j]) return;
		int tmp = nums[i];
		nums[i] = nums[j];
		nums[j] = tmp;
	}
{% endhighlight java %}
